# 🚀 Smart AI Sender Pro

Умная система для B2B-рассылок, полностью управляемая искусственным интеллектом (Google Gemini 2.5 Flash / Pro). Система берет "грязную" Excel-базу, генерирует уникальное коммерческое предложение для каждого клиента и отправляет письма, виртуозно обходя анти-спам фильтры.

## 🔥 Главные фичи (Features)

* **🧠 AI-Генерация текста:** Каждое письмо уникально. ИИ сам подставляет имена, названия компаний, перемешивает услуги и перефразирует текст строго по вашему промпту.
* **🥷 Продвинутый обход анти-спама (Anti-Fraud Bypass):**
    * **Умная ротация API-ключей:** Программа берет пул ключей и тасует их перед каждым письмом.
    * **Крипто-соль (UUID):** Невидимый уникальный ID в каждом запросе для обхода хеш-фильтров Google.
    * **Jitter (Плавающие паузы):** Случайные задержки между генерациями, имитирующие живого человека.
    * **Deep Sleep:** Автоматический "глубокий сон" на 3 минуты при перегрузке серверов (429 Rate Limit) и каскадное переключение на Pro-модель.
* **📊 Нечеткий парсинг Excel (Fuzzy Search):** Алгоритм сам найдет колонки с ФИО и названием компании по корням слов, даже если база не структурирована или разбита на разные столбцы.
* **📬 Мульти-аккаунтинг почт:** Поддержка SMTP-серверов Gmail, Yandex, Mail.ru с авто-отключением заблокированных ящиков.
* **📈 Локальная история:** Просмотр сгенерированных писем (вместе с темами) прямо в браузере и выгрузка подробного отчета обратно в Excel.

## 🛠 Стек технологий
* **Backend:** Python 3, FastAPI, Google GenAI SDK, smtplib
* **Frontend:** HTML5, CSS3, Vanilla JS, SheetJS (для работы с Excel)
